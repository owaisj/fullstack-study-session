## Node
Quick Setup: `npm init -y`
`npm i -D <pkg>` is the same as `npm install <pkg> --save-dev`

## ES6

Spread operator `...` -> spreads out an iterable object
```javascript
myFunction(...[1, 2, 3])
myFunction(1, 2, 3)
```
