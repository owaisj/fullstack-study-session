# fullstack-study-session

## Git Set-up

```bash
git clone <url>
cd <repo>
code .
npm install
```

## Node
Quick Setup (for any project):  
```
npm init -y
```  

`npm i -D <pkg>` is the same as `npm install <pkg> --save-dev`

To run the project
```bash 
npm start
```

## ES6

Spread operator `...` -> spreads out an iterable object
```javascript
myFunction(...[1, 2, 3])
myFunction(1, 2, 3)
```
